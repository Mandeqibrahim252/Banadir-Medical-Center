# Banadir Medical Center

Premier healthcare facility website located in Mogadishu, Somalia.

## Features

- Modern Healthcare Design
- Dark Mode & Light Mode
- English / Somali Language Toggle
- Fully Responsive (Mobile First)
- Appointment Booking System
- 24/7 Emergency Services
- Doctor Profiles
- Patient Testimonials
- Health Blog
- FAQ Accordion
- Contact with Google Maps
- WhatsApp Integration

## Technologies

- HTML5
- CSS3 (Custom Properties, Glassmorphism)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter + Poppins)
